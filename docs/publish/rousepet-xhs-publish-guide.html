发布正文

我把 Codex 宠物做成了久坐提醒器。 你坐得越久，它越没精神；真正离开电脑几分钟再回来，它才会回血。 它叫 RousePet。我做它的原因也很直接：传统久坐弹窗太像命令，关掉一次以后，下一次还是会顺手关掉。但看到自己的桌宠一点点蔫下来，我确实会多犹豫一下。 这是一个可以挂在笔记下方、让其他 Codex 用户直接获取的 RedSkill。拿到以后，只要对 Codex 说： “安装并立即显示 RousePet，保持我的官方 Codex 宠物完全不变。” Codex 会从包内完成安装，桌宠随后直接出现。不用另下 EXE，也不用自己找脚本。 RousePet 会只读复制你用 /hatch 创建的宠物。原来的 pet.json 和图集不会被修改，久坐记录、设置和动作副本都留在插件自己的目录。 如果你还没有 Codex 宠物，也可以给一句描述或一张参考图。新宠物只生成在 RousePet 的私有目录，不会冒充或覆盖官方宠物。 它的变化不是“坐久了突然倒地”。0 到 30 分钟保持原来的精神状态；之后依次发懒、蔫下来、明显疲惫，最后只是可逆地休息。每个阶段优先使用这只宠物原图集里已有的动作，没有对应动作就回退到相近的休息帧。 真正离开电脑几分钟再回来，它才恢复并庆祝。只移动一下鼠标，不算起身。 我最喜欢的是 Codex 接手任务的那段空窗。已经久坐时，宠物会提醒：“这段我替你盯着，出去晃一圈。”如果我真的离开五分钟，回来会记一次“听劝”。 提醒台词不是每个人都一样，也不是运行时临时调用 LLM。它会根据宠物名字、久坐阶段、当天起身次数和最近说过的话，从本地语料里选择；同时有提醒预算，不会隔几分钟念一次。 这版之前踩过不少真实的坑：龙只剩半个头、透明窗口挡桌面、点击后宠物跑偏、状态一下子全叠上去，甚至长出过很廉价的蘑菇。现在这些都被拆掉重做了。 Windows 版已完成真机验证；macOS 目前是实验性支持。元气只是桌宠互动值，不对应寿命，也不是医疗指标。 我更希望它像一只有分寸的同伴，而不是换了皮肤的健康弹窗。
