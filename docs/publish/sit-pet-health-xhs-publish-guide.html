Skill 简介

它不是一只固定形象的桌宠，而是一套“给现有 Codex 宠物加健康行为”的升级流程。Skill 只读扫描用户通过 /hatch 生成的 pet.json 与 spritesheet，把同款形象复制进 Plugin 私有目录，然后根据系统键鼠空闲时间驱动久坐、恢复和 Codex 空窗反馈。没有官方宠物时，也可以用一句描述或参考图在私有目录创建新宠物。

核心传播点：任何人的任何 Codex 宠物都能被升级；不覆盖原宠物，不要求重新画一只统一角色。

主要功能

读取用户自己的宠物

支持 CODEX_HOME/pets 和 ~/.codex/pets，多宠物可选择；源文件安装前后校验 SHA-256。

五级久坐状态

0/30/60/90/120 分钟依次为精神、发懒、蔫了、病恹恹、休息。最差状态可逆，不出现死亡。

按宠物特性映射动作

优先使用原图集的 idle、waiting、tired、sick、rest；缺少动作时回退到相近原动作，不强造统一哭泣表情。

自动判断真正起身

读取系统键鼠空闲时间。短暂离开部分恢复；连续空闲达到阈值并重新操作后，才确认一次完整起身。

Codex 空窗提醒

Codex 接手任务且用户已经久坐时，宠物提醒利用等待时间活动；真离开后记录“听劝”并庆祝。

克制的本地提醒

损友、温柔、冷幽默三种语气；所有主动提醒每小时最多 3 次，任意两次至少间隔 10 分钟。

没有宠物也能开始

提供一句描述或参考图，Skill 会生成、校验并打包到 Plugin 私有 custom-sources，不写官方宠物目录。

桌宠基本交互

支持点击庆祝、拖动、30%–250% 无级缩放、右键设置、暂停提醒、换宠物和生成分享卡。

别人如何使用

安装并立即显示我的 Codex 宠物健康版，保持官方宠物完全不变。
  1. 从小红书笔记下方获取并挂载 RedSkill。
  2. 在 Codex 里发送上面这句话。
  3. Codex 从 RedSkill 包内注册本地 Plugin，不访问 GitHub，不下载 EXE。
  4. 安装结束后桌宠立即出现；多宠物时进入选择器，没有宠物时进入创建流程。
  5. 未来第一次触发 Plugin Hooks 时，用户完成一次 Codex 官方信任审核。
  6. 右键宠物可调整尺寸、阶段节奏、离开电脑阈值、提醒语气和安静时段。
Windows 10/11 已完成真机验证；macOS 是实验性支持；Linux 暂无桌面浮窗。元气只是宠物互动值，不是寿命或医疗指标。

五阶段设计

连续活跃久坐阶段宠物表现
0–30 分钟精神保持原宠物 idle，不制造焦虑。
30–60 分钟发懒waiting 慢放，动作开始收敛。
60–90 分钟蔫了优先 tired；没有则回退 waiting/idle。
90–120 分钟病恹恹优先 sick，以动作和克制提示为主。
120 分钟以上休息/罢工优先 rest；没有则原地休息，可逆、无死亡。

数据与安全边界

  • 官方 pet.json 和 spritesheet 只读，复制前后校验哈希。
  • 复制体、健康状态、设置、自定义宠物和分享卡只写 Plugin 私有目录。
  • 不安装 EXE，不修改 Codex 配置，不上传宠物图像和健康数据。
  • Hooks 只处理开始、等待和结束事件，不读取或保存任务正文。
  • 卸载只删除自己的运行进程和私有数据，不删除 ~/.codex/pets。

发布正文

这次不是只晒一个我自己能跑的软件。 我把它整理成了一个可以挂在笔记下方、让其他 Codex 用户直接获取的 RedSkill，名字叫「Codex 宠物健康升级器」。 拿到 Skill 后，只要对 Codex 说一句： “安装并立即显示我的 Codex 宠物健康版，保持官方宠物完全不变。” Codex 会从 RedSkill 包内完成安装并立刻显示桌宠，不需要另下 EXE，也不用去外部网站找脚本。 它会只读扫描你用 /hatch 创建的官方宠物，把同款形象复制到插件自己的目录，再加入久坐、离开电脑和 Codex 任务空窗反馈。 如果还没有 Codex 宠物，也不用先研究图集格式。给它一句宠物描述或一张参考图片，Skill 会在自己的私有目录生成宠物，不会写进或覆盖官方宠物目录。 我最开始只是想把 Codex 里的小龙搬到桌面。结果做着做着，它成了我的久坐照妖镜。 坐着不动，它会从精神、发懒，一点点蔫下去；离开电脑几分钟再回来，它才重新蹦起来。 最好用的一段，是 Codex 正在替我跑任务时。宠物会冒一句：“这段没你事，去动动。”如果我真离开 5 分钟，回来还有一段庆祝。 这东西也不是一次就做顺的。中间出现过龙只剩半个头、点击后整只跑偏、透明窗口挡住桌面、病态效果像廉价蘑菇……我基本把一个桌宠能踩的坑都踩了一遍。 现在这个 Skill 可以做到： • 安装完成立即显示宠物，并建立健康动作图集 • 只读复制你已有的 Codex 宠物，不改原图集 • 没有宠物，也能用一句描述或参考图创建私有宠物 • 30/60/90/120 分钟分阶段变化，不是一秒钟突然病倒 • 系统根据键鼠空闲自动判断是否真正离开电脑 • 提醒有频率上限，不会每几分钟唠叨一次 • 不装 EXE，健康数据只存在本机 Windows 版已经完整验证；macOS 目前还是实验性支持。元气只是互动值，不代表寿命或医疗结论。 我下一步最想知道的是：你希望自己的宠物在久坐时，是安静趴下，还是直接阴阳你一句？
