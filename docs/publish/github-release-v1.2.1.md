# RousePet v1.2.1

这次更新集中处理真实使用和发布体验，不改变 v1.2.0 的私有健康动作补齐机制。

## 改进

- 设置页移除重复的“宠物大小”滑杆。尺寸继续通过桌宠右下角拖拽或滚轮实时调整，已有 `petScale` 不会被重置。
- 修正发布图中的动作帧：起身恢复使用清晰的抬手庆祝帧，安装页使用完整微笑站立帧。
- 设置页宣传图改为实际窗口重新截图，不再展示已经取消的尺寸卡。
- 小红书正文改为更短的第一现场叙述，标签改为工作场景、大众效率词、产品精准词与 `#REDSkill` 混搭。

## 仍然保持

- 自动判断宠物是否明确提供 `tired / sick / rest`；只为缺失项生成私有六帧动作。
- 生成前立即使用安全回退动作显示宠物，质检通过后原子切换。
- 官方 `pet.json` 和 spritesheet 始终只读，生成内容只写入 RousePet Plugin 私有目录。
- 不安装独立 EXE，不读取或保存 Codex 任务正文。
