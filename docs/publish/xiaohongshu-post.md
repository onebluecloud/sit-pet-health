## 正文

我把 Codex 里孵出来的宠物，做成了一只会陪我“坐蔫”的久坐提醒器。

坐着不动时，它不会突然倒地，而是从精神、发懒、蔫了、病恹恹到休息，一点点慢下来。真正离开电脑几分钟再回来，它才回血庆祝；只晃一下鼠标不算。

最好用的是 Codex 跑任务的空窗。它接手后，宠物会提醒我：“这段没你事，去动动。”我真离开五分钟，回来还会多记一次“听劝”。

它叫 RousePet，是我做的原创 RedSkill。拿到后对 Codex 说：

“安装并立即显示 RousePet，保持我的官方 Codex 宠物完全不变。”

它会只读复制你的 /hatch 宠物。缺少 tired、sick 或 rest 时，先用原动作立即显示，再只在私有目录补齐缺失动画；官方 pet.json 和图集不会改。没有宠物，也可以用一句描述或参考图创建一只私有宠物。

Windows 已真机验证，macOS 目前是实验性支持。它不是医疗工具，元气只是互动值。

你会为了不让自己的宠物蔫掉，真的站起来五分钟吗？
