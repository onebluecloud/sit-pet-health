## 正文

我本来只想把 Codex 里的小龙搬到桌面。

结果做着做着，它成了我的久坐照妖镜。

坐着不动，它会从精神、发懒，一点点蔫下去；离开电脑几分钟再回来，它才重新蹦起来。

最好用的一段，是 Codex 正在替我跑任务时。宠物会冒一句：“这段没你事，去动动。”如果我真离开 5 分钟，回来还有一段庆祝。

这东西也不是一次就做顺的。中间出现过龙只剩半个头、点击后整只跑偏、透明窗口挡住桌面、病态效果像廉价蘑菇……我基本把一个桌宠能踩的坑都踩了一遍。

现在收敛成了一个 RedSkill：

• 只读复制你已有的 Codex 宠物，不改原图集  
• 没有宠物，也能用一句描述或参考图创建私有宠物  
• 30/60/90/120 分钟分阶段变化，不是一秒钟突然病倒  
• 系统根据键鼠空闲自动判断是否真正离开电脑  
• 提醒有频率上限，不会每几分钟唠叨一次  
• 不装 EXE，健康数据只存在本机

Windows 版已经完整验证；macOS 目前还是实验性支持。元气只是互动值，不代表寿命或医疗结论。

Skill 挂在笔记下方。拿到后对 Codex 说：

“安装并立即显示我的 Codex 宠物健康版，保持官方宠物完全不变。”

我下一步最想知道的是：你希望自己的宠物在久坐时，是安静趴下，还是直接阴阳你一句？

## 配图顺序

1. 封面：健康宠物 + “你不起来，它就先蔫了”。
2. 五阶段：精神、发懒、蔫了、病恹恹、休息，强调渐进而非死亡。
3. Codex 空窗：任务运行时的气泡提醒。
4. 起身反馈：离开五分钟后庆祝、元气恢复。
5. 设置页：宠物、尺寸、久坐节奏、语气和安静时段。
6. 数据边界：官方宠物只读、数据本地、不读取任务正文。
7. 安装页：RedSkill 获取后的一句安装提示词。
